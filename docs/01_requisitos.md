## Sistema Operativo seleccionado

Para la propuesta final de esta estación de trabajo para ingeniería se ha seleccionado **Windows 11 Pro de 64 bits** como sistema operativo.

### Justificación técnica

- **Compatibilidad nativa:** Proteus Design Suite está diseñado específicamente para Microsoft Windows. Su ejecución en macOS o Linux requiere máquinas virtuales o capas de compatibilidad, lo que puede reducir el rendimiento y generar inconvenientes de compatibilidad.

- **Aprovechamiento del hardware:** Windows 11 ofrece un buen soporte para procesadores multinúcleo y arquitecturas híbridas, además de compatibilidad con las versiones más recientes de DirectX y los controladores de tarjetas gráficas dedicadas, lo que favorece el rendimiento de Autodesk Fusion.

- **Estabilidad para uso profesional:** Windows 11 cuenta con amplia compatibilidad con los controladores oficiales del hardware seleccionado, facilitando la estabilidad del sistema, las actualizaciones y el soporte de los fabricantes.

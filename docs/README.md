# 💻 Proyecto: Estación de Trabajo para Ingeniería

## Trabajo Práctico

### Documentación Técnica de una Estación de Trabajo para Ingeniería

Este proyecto presenta el diseño y la selección de una estación de trabajo destinada al desarrollo de actividades de ingeniería, considerando los requerimientos de software como **Autodesk Fusion**, **Proteus Design Suite**, **Visual Studio Code** y **Adobe Acrobat Reader**.

La documentación se encuentra organizada en las siguientes secciones:

-  **Requisitos del sistema**
-  **Justificación técnica del hardware seleccionado**
-  **Presupuesto estimado**
-  **Conclusiones**
